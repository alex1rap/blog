## Installation
- Create database and DBMS user
- Copy and configure file `.env`:
```shell
cp .env.example .env
```
- Install dependencies:
```shell
composer install
npm install
```

## Starting
- Build frontend:
```shell
npm run dev
```
- Start backend:
```shell
php artisan serve
```

## Open in browser
By default, project address is: http://localhost:8000.
